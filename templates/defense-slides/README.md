# Beamer 工程项目答辩模板

## 编译

```bash
latexmk -xelatex -interaction=nonstopmode -halt-on-error main.tex
```

输出为 16:9 PDF，可直接用于投屏或导入支持 PDF 的演示软件。

## 已包含页面

- 标题页和目录
- 项目背景、目标和关键指标
- 系统架构图
- 控制算法和安全策略
- 状态机
- 测试方案和结果表
- 创新点、局限和下一步
- 结束页

## 使用建议

- 每页只保留一个主要结论，正文尽量控制在 40–60 个汉字以内。
- 用实物照片、原理图局部和真实曲线替换示例内容时，确认图片版权和个人信息。
- 结果页必须标注单位、工况、仪器和数据来源。
- 模板不是任何学校或竞赛的官方答辩格式。
